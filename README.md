# \# Інформаційна система управління бібліотекою

# 

# Цей репозиторій містить практичні роботи з дисципліни «Системний аналіз та проектування інформаційних систем». Проєкт присвячений розробці архітектури та аналізу сучасної системи автоматизації бібліотечних процесів.

# 

# \## Огляд проєкту

# Метою системи є автоматизація ключових робочих процесів бібліотеки: реєстрація книг, управління базою читачів, контроль видачі та повернення примірників. Проєкт охоплює шлях від збору вимог до опису механізмів масштабування та безпеки.

# 

# \## Структура репозиторію

# Усі аналітичні матеріали та описи розділені за темами й знаходяться у папці `content/`:

# \- \*\*Розділи 1-7:\*\* Аналіз вимог, User Stories, архітектура та модель даних.

# \- \*\*Розділ 8:\*\* Обґрунтування вибору реляційної бази даних (PostgreSQL).

# \- \*\*Розділ 9:\*\* Аналіз можливостей масштабування (вертикальне/горизонтальне).

# \- \*\*Розділ 10:\*\* Система безпеки (JWT-токени, рольова модель доступу RBAC).

# \- \*\*Розділ 11:\*\* Перспективи розвитку та майбутні модулі.

# \- \*\*Розділ 12:\*\* Методологія розробки (Agile/Scrum).

# \- \*\*Розділи 13-16:\*\* Документування, системні цикли та точки впливу на систему.

# 

# \## Технологічний стек (Концептуальний)

# \- \*\*База даних:\*\* PostgreSQL.

# \- \*\*Безпека:\*\* JWT Authentication, BCrypt для хешування паролів.

# \- \*\*Методологія:\*\* Agile (Scrum).

# \- \*\*Нотації:\*\* UML.
